# LLM Config

## Overview
This is a repo that abstracts all LLMs into components config for convenient management.

## Features

- Manual config as components for LLM.
- Support to parse config from file.

## Requirements

- sqlalchemy

## Installation

```bash
pip install git+https://github.com/IS-Model-Framework/llm-config.git
```
